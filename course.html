About Course

Visual Interface- No-code ပလပ္ေဖာင္းမ်ားသည္ အသုံးျပဳသူမ်ားအား အပလီေကးရွင္းကို အျမင္အာ႐ုံျဖင့္ ဒီဇိုင္းထုတ္ႏိုင္ေစမည့္ graphical user interface (GUI) ကို ပံ့ပိုးေပးပါသည္။ အသုံးျပဳသူမ်ားသည္ ျဒပ္စင္မ်ားကို ဆြဲခ်ျခင္း၊ ေဒတာေမာ္ဒယ္မ်ားကို သတ္မွတ္ျခင္းႏွင့္ ကုဒ္မေရးဘဲ အသုံးျပဳသူၾကားခံမ်ားကို ဖန္တီးႏိုင္သည္။

အစိတ္အပိုင္း စာၾကည့္တိုက္မ်ား- ဤပလပ္ေဖာင္းမ်ားသည္ အသုံးျပဳသူမ်ား ၎တို႔၏ အပလီေကးရွင္းမ်ားတြင္ ေပါင္းစည္းႏိုင္သည့္ ႀကိဳတင္တည္ေဆာက္ထားေသာ အစိတ္အပိုင္းမ်ားႏွင့္ ေမာ္ဂ်ဴးမ်ားကို ေပးေဆာင္ပါသည္။ ဤအစိတ္အပိုင္းမ်ားသည္ UI ျဒပ္စင္မ်ား (ခလုတ္မ်ား၊ ေဖာင္မ်ား၊ မီႏူးမ်ား) မွ ရႈပ္ေထြးေသာလုပ္ေဆာင္ႏိုင္စြမ္းမ်ား (ေဒတာေဘ့စ္ေပါင္းစည္းမႈ၊ အေထာက္အထားစိစစ္မႈ၊ ေဒတာျမင္ေယာင္မႈ) တို႔အထိ ကြဲျပားႏိုင္သည္။

အလုပ္အသြားအလာ အလိုအေလ်ာက္လုပ္ေဆာင္ျခင္း- ကုဒ္မရွိေသာ ပလပ္ေဖာင္းမ်ားတြင္ မၾကာခဏဆိုသလို အလုပ္အသြားအလာ အလိုအေလ်ာက္လုပ္ေဆာင္ႏိုင္စြမ္းမ်ား ပါဝင္ၿပီး သုံးစြဲသူမ်ားသည္ အစပ်ိဳးမႈမ်ား သို႔မဟုတ္ ျဖစ္ရပ္မ်ားအေပၚ အေျခခံ၍ လုပ္ေဆာင္မႈမ်ား၏ ဆက္တိုက္မ်ားကို သတ္မွတ္သတ္မွတ္ႏိုင္ေစပါသည္။ ၎သည္ ထပ္ခါတလဲလဲ လုပ္ေဆာင္စရာမ်ားကို အလိုအေလ်ာက္လုပ္ေဆာင္ျခင္း သို႔မဟုတ္ ကုဒ္မပါဘဲ ရႈပ္ေထြးေသာ လုပ္ငန္းလုပ္ငန္းစဥ္မ်ားကို ဖန္တီးႏိုင္ေစပါသည္။

ေဒတာေပါင္းစည္းျခင္း- ကုဒ္မရွိေသာ ပလပ္ေဖာင္းအမ်ားအျပားသည္ ျပင္ပေဒတာရင္းျမစ္မ်ား၊ API မ်ားႏွင့္ ေဒတာေဘ့စ္မ်ားႏွင့္ ေပါင္းစည္းမႈကို ပံ့ပိုးေပးသည္။ ၎သည္ အသုံးျပဳသူမ်ားအား မတူညီေသာ စနစ္မ်ားမွ ေဒတာမ်ားကို ဆြဲထုတ္ကာ ႀကိဳးကိုင္ကာ ၎တို႔၏ အပလီေကးရွင္းမ်ားအတြင္း ကုဒ္မေရးဘဲ ျပသႏိုင္ေစပါသည္။

About Course

ပ႐ိုဂရမ္းမင္းနယ္ပယ္သည္ ကြန္ပ်ဴတာပ႐ိုဂရမ္မ်ား သို႔မဟုတ္ ေဆာ့ဖ္ဝဲလ္မ်ားကို ဖန္တီးျခင္း၊ အေကာင္အထည္ေဖာ္ျခင္းႏွင့္ ထိန္းသိမ္းျခင္းတို႔ကို လႊမ္းၿခဳံထားသည္။ ပ႐ိုဂရမ္းမင္းတြင္ တိက်ေသာအလုပ္မ်ားလုပ္ေဆာင္ပုံ သို႔မဟုတ္ ျပႆနာမ်ားကို ေျဖရွင္းနည္းကို ကြန္ပ်ဴတာအား ေျပာျပေသာ ကုဒ္ဟု လူသိမ်ားေသာ စာေရးၫႊန္ၾကားခ်က္မ်ား ပါဝင္ပါသည္။

အစိတ္အပိုင္း စာၾကည့္တိုက္မ်ား- ဤပလပ္ေဖာင္းမ်ားသည္ အသုံးျပဳသူမ်ား ၎တို႔၏ အပလီေကးရွင္းမ်ားတြင္ ေပါင္းစည္းႏိုင္သည့္ ႀကိဳတင္တည္ေဆာက္ထားေသာ အစိတ္အပိုင္းမ်ားႏွင့္ ေမာ္ဂ်ဴးမ်ားကို ေပးေဆာင္ပါသည္။ ဤအစိတ္အပိုင္းမ်ားသည္ UI ျဒပ္စင္မ်ား (ခလုတ္မ်ား၊ ေဖာင္မ်ား၊ မီႏူးမ်ား) မွ ရႈပ္ေထြးေသာလုပ္ေဆာင္ႏိုင္စြမ္းမ်ား (ေဒတာေဘ့စ္ေပါင္းစည္းမႈ၊ အေထာက္အထားစိစစ္မႈ၊ ေဒတာျမင္ေယာင္မႈ) တို႔အထိ ကြဲျပားႏိုင္သည္။

အလုပ္အသြားအလာ အလိုအေလ်ာက္လုပ္ေဆာင္ျခင္း- ကုဒ္မရွိေသာ ပလပ္ေဖာင္းမ်ားတြင္ မၾကာခဏဆိုသလို အလုပ္အသြားအလာ အလိုအေလ်ာက္လုပ္ေဆာင္ႏိုင္စြမ္းမ်ား ပါဝင္ၿပီး သုံးစြဲသူမ်ားသည္ အစပ်ိဳးမႈမ်ား သို႔မဟုတ္ ျဖစ္ရပ္မ်ားအေပၚ အေျခခံ၍ လုပ္ေဆာင္မႈမ်ား၏ ဆက္တိုက္မ်ားကို သတ္မွတ္သတ္မွတ္ႏိုင္ေစပါသည္။ ၎သည္ ထပ္ခါတလဲလဲ လုပ္ေဆာင္စရာမ်ားကို အလိုအေလ်ာက္လုပ္ေဆာင္ျခင္း သို႔မဟုတ္ ကုဒ္မပါဘဲ ရႈပ္ေထြးေသာ လုပ္ငန္းလုပ္ငန္းစဥ္မ်ားကို ဖန္တီးႏိုင္ေစပါသည္။

ေဒတာေပါင္းစည္းျခင္း- ကုဒ္မရွိေသာ ပလပ္ေဖာင္းအမ်ားအျပားသည္ ျပင္ပေဒတာရင္းျမစ္မ်ား၊ API မ်ားႏွင့္ ေဒတာေဘ့စ္မ်ားႏွင့္ ေပါင္းစည္းမႈကို ပံ့ပိုးေပးသည္။ ၎သည္ အသုံးျပဳသူမ်ားအား မတူညီေသာ စနစ္မ်ားမွ ေဒတာမ်ားကို ဆြဲထုတ္ကာ ႀကိဳးကိုင္ကာ ၎တို႔၏ အပလီေကးရွင္းမ်ားအတြင္း ကုဒ္မေရးဘဲ ျပသႏိုင္ေစပါသည္။
